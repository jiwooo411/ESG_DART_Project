# research_log/

초기 분석(N=210)부터 최종 분석(N=381)까지의 주요 분석 및 검증 과정을 기록하였다.

## 구성

| 폴더 | 내용 |
|---|---|
| `reports/` | 단계별 진행 보고서 (수집 → 전처리 → 검증 → 최종 보고서) |
| `recovery_pilot/` | 초기 표본 복구 단계의 진단·검증 자료 |
| `recovery_N381/` | 전수 표본(381) 수집 검증 자료 |
| `notebooks_archive/` | 최종 분석 노트북 이전 단계의 작업 노트북 |

최종 분석 결과와 결론은 루트 `README.md`와 `paper/`, `notebook/`를 참조할 것 — 이 폴더는 그 결과에 도달하기까지의 중간 기록이다.
