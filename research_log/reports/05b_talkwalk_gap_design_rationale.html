말–실행 격차(Talk–Walk Gap) 기업 유형 진단

왜 이 표본만 가능한가. 선행연구는 "잘 쓰면 등급↑"라는 평균효과에서 멈추거나(#1·#3), 격차의 'walk'로 배출량·환경위반을 쓴다(#5·#6). 본 프로젝트는 ⓐ사전 확장 어휘강도(talk), ⓑ독립기관 KCGS 등급(walk-proxy), ⓒ최대 교란인 분량을 모두 보유 — 세 조건이 동시에 갖춰진 표본은 드물다.

분석 파이프라인 · 5단계

1
어휘강도 추출
차원별 expanded_score_E/S/G = 기업의 ESG '말' 신호
2
분량 잔차화
~ log_n_tokens에 회귀한 잔차 → 분량 걷어낸 순수 강조도
3
z-표준화
talk 잔차와 grade를 같은 척도로 정렬
4
격차 점수
gap = z(talk) − z(grade) 계산 (Delmas·Kim-Lyon 계열)
5
사분면 + 지속성
2×2 유형 분류 · 3개 연도 반복 여부로 구조적/일시적 구분
gapd = z( talkdresid ) − z( graded )   |   talkdresid = resid( expanded_scored ~ log_n_tokens )
gap > 0 말이 등급보다 앞섬 (과잉) gap < 0 등급이 말보다 앞섬 (과소) d ∈ {E, S, G}

유형별 경영 제언 · 이 알파의 결과물

과잉 공시형over-talk · gap>0

리스크. 언어가 등급을 앞서면 평가사·규제기관의 할인(discount) 및 greenwashing 조사 대상이 되기 쉽다.

→제언: 서술을 검증 가능한 정량 목표·실적(배출량 수치, 안전교육 이수율, 사외이사 비율)으로 뒷받침하라.
과소 공시형under-talk · gap<0

리스크. 등급은 좋은데 보고서 언어가 약함 = 인식 격차(IR gap). KCGS가 사업보고서를 평가 인풋으로 쓰므로 저평가·시장 미반영 손실 가능.

→제언: 실질 성과를 사업보고서 II·IV·VI에 구체적으로 서술해 평가·시장 인식을 회수하라.
전사(全社) 차원통념 반증

"ESG 어휘를 많이·강하게 쓰면 곧 성과"라는 통념을 데이터로 반증(#7).

→제언: 공시 KPI를 '양(量)'에서 '검증가능성·정합성'으로 전환하라.
⚠︎한계. KCGS 등급을 'walk'의 대리로 쓰지만 등급 자체가 noisy(#10·#12)하고 일부 사업보고서를 인풋으로 한다(동어반복·역인과 위험) → 격차는 인과적 greenwashing 측정이 아닌 진단 신호로만 해석. 3개 연도는 지속성 판단에 짧음.

기업내(within-firm) 변화 분석

동기. 기준 분석은 횡단면(pooled). Bingler(#4)는 "TCFD 지지 선언이 실제 공시 증가로 이어지지 않음" — 즉 시간에 따른 말-실행 분리를 보였다. 3개 연도 패널로 횡단면 G 신호가 기업 간(구조적)인지 기업 내(연도 대응)인지 분해한다.

Δ g_grade  ~  Δ expanded_G  +  Δ log_n_tokens   (1차 차분 / 기업 FE)

예상·시사점: 등급이 sticky하고 within 분산이 작아 신호가 between이면 → ESG 언어는 연도별 공시 전술이 아닌 안정적 기업 유형을 반영. 제언: 단기 워딩 변경보다 구조적 거버넌스 개선이 등급에 유효. (단, 3개 연도·낮은 within 분산 → 검정력 약함 명시.)

왜 1순위는 Talk–Walk Gap인가

네 가지 이유에서 메인 알파로 권장한다.

a프로젝트가 강조한 cheap-talk 프레이밍을 기업 단위 진단·처방으로 끌어올려 §5 결론의 자연스러운 다음 단계.
b추가 수집 없이 기존 PANEL_DF로 즉시 실행 가능.
c과잉/과소 두 군집에 명확한 경영 제언이 나와 평가표의 '알파'·'해석과 한계'를 동시 충족.
d선행연구 빈틈(firm-level 진단 부재)을 정확히 겨냥.
